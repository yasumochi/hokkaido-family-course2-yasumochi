# 北海道 旅のしおり ─ ページ制作テンプレート

札幌大交流会（2026年7月10〜12日）の旅のしおりサイト、
その**1ページ分**を作るためのテンプレートです。
リベシティ北海道支部の有志企画でつかいます。

## 使い方（かんたん版）

支部から配られた「**スタートガイド**」の呪文を、Claude Code に貼るだけ。
あとは Claude が、ページ作りからインターネット公開まで案内してくれます。
むずかしいことを覚える必要はありません。

## ファイルの説明

| ファイル | これは何 | さわる？ |
|---|---|---|
| `index.html` | ページの中身。ここを書き換えます | ✅ ここを編集 |
| `images/` | 写真を入れるフォルダ | ✅ 写真を追加 |
| `style.css` | デザイン（見た目）の共通ファイル | ❌ さわらない |
| `CLAUDE.md` | Claude 向けのルール | ❌ さわらない |

## 完成したら

Cloudflare Pages にデプロイ（公開）して、公開URL（`〇〇.pages.dev`）を取得します。
そのURLを支部の担当者に送ってください。
旅のしおりトップページに、あなたのページへのボタンが追加されます。
