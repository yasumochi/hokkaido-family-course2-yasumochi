# 📐 こうきママさんサイト「スポット詳細カード」標準レイアウト

> このレイアウトは、こうきママさんサイトで各スポットを紹介するカードの共通フォーマット。
> やすもちのスポットカードもこの順番・項目で作成する。

## カード構成（上から順に）

| # | 項目 | 内容 |
|---|---|---|
| **1** | **施設名** | スポットの名前（例：新千歳空港） |
| **2** | **施設の写真（複数）** | そのスポットの写真。複数枚OK（カルーセル） |
| **3** | **見出し** | キャッチーな一文。お子様連れ目線の実用Tips入り |
| **4** | **オススメポイント欄** | 「○○のオススメポイント」＋担当者名。詳しい説明文（★で小見出し） |
| **5** | **乳幼児設備 有無欄** | 授乳室・おむつ替え台・ベビーカー貸出。あれば各MAPへリンク |
| **6** | **○○の声** | 担当者アイコン＋ひとことコメント（体験談） |
| **7** | **予算の欄** | 料金目安 |
| **8** | **DATA** | 住所・電話・営業時間（調べて記載）＋mapリンク |
| **9** | **公式ページへ遷移** | 「詳しく見る」ボタン → 施設公式サイト |

## 制作時のルール
- 見出し(#3)は**スポットの内容と一致**させる（他コースの内容を混ぜない）
- オススメポイント(#4)は担当者名（やすもち／なのぱぱ等）を明記
- 乳幼児設備(#5)は「あり/なし」を明確に。ありなら詳細ページへ
- ○○の声(#6)は担当者の実体験・ひとこと
- DATA(#8)は公式サイトで最新情報を確認してから記載
- 公式ページリンク(#9)は必ず公式の正しいURL
